README

暗闇の高所で股間が寒くなる恐怖感を体感せよ！ 階段を駆け下り、時には駆け上がり、指定された数の鍵を集めてから、遥か下にある出口を目指せ！ 点滅している階段の踊り場は、ジャンブで上の階段にスルーアップできるぞ！ たとえどんな高さから飛び降りても、下に階段の踊り場があれば着地が可能。だが、欲張りすぎて着地に失敗すると…奈落の底へと落ちてしまうのでくれぐれも注意！ 暗闇を照らすことができる懐中電灯のバッテリーには限りがある。途中でバッテリーを拾い、少しでも長く暗闇を照らせるようにしろ！
