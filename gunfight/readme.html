遊び方

Gun Fight Game へようこそ！このゲームは、シンプルな操作で楽しめるアクションシューティングゲームです。 様々なマップで敵を倒し、最高のスコアを目指しましょう。

基本的な操作方法

  • 移動: [W], [A], [S], [D] キー
  • 攻撃: マウスの左クリック
  • リロード: [R] キー
  • 武器切り替え: [1], [2], [3] キー

マップエディターについて

このゲームには、自分だけのオリジナルマップを作成できる強力なマップエディターが付属しています！ 「Map_editer」ボタンからエディターを起動し、自由に地形や障害物、敵の配置などをデザインできます。 作成したマップで友達とプレイするもよし、公開して世界中のプレイヤーに挑戦してもらうもよし、遊び方は無限大です。

詳しい操作方法や機能については、ゲーム内のチュートリアルやエディターのヘルプをご確認ください。
